# Budget Wise Web App

A lightweight personal budgeting app that helps users track income, expenses, and savings goals directly in the browser.

## Why it matters

Financial literacy tools make budgeting more practical. This app demonstrates frontend logic, local persistence, and a clean product workflow.

## Run

Open `index.html` in a browser.

## Features

- add income or expenses
- see current balance
- track savings target progress
- persist entries in `localStorage`
